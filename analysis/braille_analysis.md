# Braille analysis

## Summary
1. Основные конкуренты – на платформах для мобильных устройств, на ПК обучение Брайлю не распространено или поставляется в комплекте с физическими устройствами ввода Брайлевского текста.
2. Большинство приложений создано для зрячих людей, что делает наш проект чуть более уникальным.
3. Приложений с русским алфавитом также немного.
4. Фичи:  
* Голосовое управление приложением
* 	Запоминание букв с помощью ассоциаций, указания на сходство обычного написания буквы и Брайлевского
* 	Режим словаря
* 	Ввод с помощью смахиваний
* 	Отдельный курс по цифрам с встроенным калькулятором
* 	Отдельные компоненты приложения можно будет доскачать в отдельном активити (не знаю, понравится ли Михаилу, но это сделает основное приложение более легким)
5. Идеи для монетизации
*	подписка на определенный срок
*	платные надстройки
*	единоразовая покупка за версию без рекламы
